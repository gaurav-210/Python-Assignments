
# A quick Revision of the used commands.
 
1. `git init` - Create an empty git repository
 
2. `git add .` - Add all files to the next commit/version
 
3. `git status` - Gives the status for thr files which are tracked,untracked and modified.
 
4. `git commit -m "Your meaningful message here"` - Create a commit/version of the proejct

5. `git log` - See commit/version history of the project

6. `git remote add origin 'Paste your link for remote repo here'` to add the link btw remote repo and local repo.
 
7. `git push origin master` - To push your code ,your project project file to Remote repo from local repo.
